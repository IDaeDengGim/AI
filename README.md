영화 시놉시스가 아래와 같은 내용을 담고 있으면 예외 발생함.

- HARM_CATEGORY_SEXUALLY_EXPLICIT: 성적으로 노골적인 콘텐츠
- HARM_CATEGORY_HATE_SPEECH: 증오 발언
- HARM_CATEGORY_HARASSMENT: 괴롭힘
- HARM_CATEGORY_DANGEROUS_CONTENT: 위험한 콘텐츠
